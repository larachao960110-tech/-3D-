## ✨ 練習重點技能

### 🧱 3D 建模（Modeling）
- 使用 **Edit Mode** 進行頂點、邊、面操作
- 透過 **Subdivision Surface** 修改器實現布料圓潤感
- 比例式編輯（Proportional Editing）調整枕頭自然形變

### 🎨 材質與貼圖（Shading）
- 使用 **Principled BSDF** 設定布料質感
- 調整 Roughness / Specular 模擬織物表面
- Shader Node 節點編輯入門

### 💡 燈光與渲染（Rendering）
- **Cycles** 或 **EEVEE** 渲染引擎設定
- 三點打光（Three-point Lighting）基礎應用
- 輸出高品質渲染圖
